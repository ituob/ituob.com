Choosing a Publication ID

The application allows any text as an publication ID, but you should take care using consistent well-formatted IDs to ensure ITU OB database is properlly formatted.

  • Publication ID can include uppercase letters, numbers, and underscores.
  • If the publication is associated with an ITU recommendation, the ID should include a recommendation code.
  • Here are some examples of well-rounded publication IDs:
    • E212_MNC
    • X121_DCC
    • BUREAUFAX

This guide will show how to create an “amendment” message in a given ITU OB Edition.

It is assumed that you know the title or ID of the publication you will amend, and you will have opened Issue Editor for your desired OB Edition.
