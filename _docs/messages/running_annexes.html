In the Message Editor pane on the right, two tabs should be visible; “Running Annexes” and “Extra Publications”.

  • Running Annexes tab contents are populated automatically based on publications annexed to preceding OB issues.
  • Extra Publications requires items to be added manually.
icon: fas fa-info-circle intent: primary - step: | Switch to the Extra Publications tab, and use the circular plus button to add each publication that normally appears under part B. of Lists Annexed section in each ITU OB issue. notes: - text: |

Only enter a short Publication code for each item.

Publication codes to use:
Publication name Publication code
List of ITU Carrier Codes (ITU-T Rec. M.1400 (03/2013)) M1400_ICC
Bureaufax Table (ITU-T Rec. F.170) BUREAUFAX
List of recognized operating agencies (ROAs) ROA

After clicking the Add Publication button, changes will be recorded locally. icon: fas fa-info-circle intent: primary - step: | To remove an incorrectly added item, click the red cross button on the right. ---

This guide explains how to add the Lists Annexed message to ITU OB Edition using the ITU OB Editor app.

Assuming the Issue Editor is open for the OB Edition for the current project and the individual has the general knowledge for adding messages.
